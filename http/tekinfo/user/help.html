用户

基本概念
  • 用户

           使用计算机信息系统的任何人都需要一个身份，这个身份在系统中被称为用户。

  • 基本内容

           用户通常包括用户编码、用户名字以及在系统中的基本账户信息。

  • 更多说明

           对于 统一资源平台 ，一般采用 诠证通 系统作为用户登录的认证系统。 因此，用户的登录安全等信息，存放在ipass系统中。当用户使用基于 统一资源平台 开发的应用系统中，可以授权应用系统取得 诠证通 存放的基本账户信息，作为该应用系统中的个人账户信息。

  • 用户操作

           个人用户可以在个人管理界面中，修改自己的头像、用户名等账户信息。 这些信息可以和用户存放在 诠证通 系统中的完全不同，这样方便用户在不同应用系统中，以不同的面貌出现。
           个人用户也可以通过查看自己的登录时间和ip发现问题。并将问题报告给管理员。

管理员指南
  • 管理

           管理员可以在管理员后台界面管理用户信息。可以更改，指派用户的安全等级等内容。

  • 安全等级

           通过安全等级的设定，管理员可以设置用户的角色。包括：

    • 超级管理员：系统中唯一最高权限用户;
    • 管理员：负责系统管理的用户，不能设置其他管理员的信息，通常情况下，管理员主要负责用户的管理。
    • 审核员：主要负责对象的内容更改，和处理用户申诉流程等。
    • 注册用户：正常的系统使用者。
  • 状态

           用户状态可以处于申请；有效和无效。用户状态的更改由管理员负责。管理员可以禁止用户（设定为无效）进入系统，也可以禁止用户一定时间内有效或者无效。
           统一资源平台也为管理也提供了更多的统计、报表等辅助工具。
