添加 对象目录分类 配置
目录树配置 [catalog_config]
目录树配置，定义指定对象“只允许定义公共目录树”或“只允许定义私有目录树”。默认 只允许定义公共目录树。
基值 [catalog_base]
基值，为编码的前缀。例如编码 ZX020200 基值为 ZX；编码 020200 基值 为 空""。基值必须是数字字母组合（包含特殊符号），以字母结尾，可以为空值。
深度 [catalog_depth]
深度，表示编码的分几级的级数。例如编码 ZX020200 每2个数字为一级的话，其深度为 3。深度必须是大于 0 的整数值。
位数 [catalog_digit]
位数，表示编码每一级的数字位数。例如编码 ZX020200 深度为3的话，其每一级的位数为 2。位数必须是大于 0 的整数值。
步长 [catalog_step]
步长，表示编码同一级中每一次增减的跨度。例如编码 ZX020200 步长为2，则同级中下一个编码值为 ZX020400。步长必须是大于 0 的整数值。
编码类型 [catalog_tree]
编码类型，分“杜威”和“默认”两种。杜威类型，编码的缺省为会用“0”来补全；默认类型，缺省为空缺。 例如，基值为 ZX、深度为 3、位数为 2、步长为 2 的编码，第一级中某一编码，杜威类型形式为 ZX040000，对应的默认类型形式为 ZX04；第二级中某一编码，杜威类型形式为 ZX020400，对应的默认类型形式为 ZX0204。
