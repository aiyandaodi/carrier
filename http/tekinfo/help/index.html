统一资源平台

统一资源平台

         统一资源平台 是由 北京易商迅达科技有限公司［Tekinfo Communication Co., Ltd.］自主开发的，并拥有独立知识产权的互联网+中间件系统。统一资源平台是介于J2EE容器和操作者接口之间的，可生长的应用开发平台。

         统一资源平台 是采用JAVA语言开发的互联网应用的基础平台，可以任意定义不同应用"对象"。 具有高度适配性和扩展性，配置知识管理以及通讯服务等协同办公应用，可以按需定制办公自动化、电子商务、智慧社区、健康管理等多种应用。

研发目标

         作为行业内领先的信息化技术按需定制开发商， 北京易商迅达科技有限公司 面向不同行业开发各自的应用，这些应用，包括面向过程的工业管理、流程管控，也包括面向协同办公等各类具体的应用场景。 所有这些应用都构筑在不同的软件、硬件架构下，唯一相同的是，这些应用都是基于网络环境。 为了面向不同企业快速搭建一个健壮且具有扩展性的，适应不同环境的系统，迫切需要一个可以自主生成的中间件平台，我们希望这个平台可以：

    • 兼容不同的操作系统;
    • 支持主流行业标准数据库；
    • 具有简洁易理解的定义方式；
    • 面向对象的开发语言；
    • 支持行业标准，可以方便的扩展；
    • 丰富的基础应用模块；
    • 适应不同类型的应用终端。

         从2000年开始，历经多次迭代， 统一资源平台 已经发展到第九代。同时，在不断的研发过程中，也形成了面向政务、商务的多样化应用实例。

主要特点

         统一资源平台 兼容行业标准并具有高可扩展性。

  • XML定义平台的基础是由一个核心的XML解释器组成。对象的数据库定义，对象的属性和方法都可以使用XML进行定义，然后由程序自动解释执行。
  • 操作系统 可运行于Windows、MacOS、Unix、Linux等多种主流操作系统上。
  • 数据库所有的应用都支持主流的关系型数据库：Mariadb、MS-SQL、MySql、Oracle、PostGresql等。
  • JAVA采用原生JAVA，高效简洁。原生代码更易理解，便于开发者的系统重构或二次开发。
  • J2EE平台也同时支持J2EE的各种标准Servlet容器。包括Glassfish、JBOSS、Resin、Tomcat等
未来发展

         统一资源平台 从2000年发布第一版以来，支持B/S应用。在后续的版本中，第六版至第八版，加入了支持C/S框架，这样， 统一资源平台 成为国内少有的同时支持C/S和B/S模式的开发平台

         从2015年开始，随着国内互联网+应用的一枝独秀， C/S的企业应用日渐式微，重新规划和设计的第九代版本不再支持C/S应用模式，但增加了对移动等多样化终端设备的支持。 为了更好适应互联网+的应用开发，新的版本加入了基于互联网的统一认证，社交管理等模块，为企业业务互联网化奠定了基础。

         新一代 统一资源平台 ，为了帮助企业的业务应用和加强互联网在线运营，更多的支持了社交应用。 同时，支持移动端APP的应用开发，和开放社交（例如微信公众号）的嵌入式支持。

基础架构

         统一资源平台 基础架构示意图。

统一资源平台 采用了多层架构模式。 从开发角度和应用角度来看，多层架构适合群体开发，协同工作，使效率倍增。

  1. 开发人员可以只关注整个结构中的其中某一层；
  2. 可以很容易地用新的实现来替换原有层次的实现；
  3. 可以降低层与层之间的依赖；
  4. 有利于标准化；
  5. 利于各层逻辑的复用。
  6. 扩展性强。
  7. 安全性高。
  8. 项目结构更清楚，分工更明确，有利于后期的维护和升级

应用架构

         统一资源平台 应用架构示意图。

新一代的 统一资源平台 ， 可以更稳定更便捷地支持B/S模式的应用开发和移动终端的APP开发。

行业应用

         统一资源平台 从2000年发布第一版以来，历经多次迭代，已经在政府、企事业单位等多种信息系统环境下应用，并获得成功。以下仅仅是部分应用。

  • 办公OA 在部委及多个政府单位应用，具有强大的协同办公及业务处理能力。
  • 电子取证 应用于公安等多个行业的电子取证综合平台，完全按照规范的取证流程进行，保证取证合法性。
  • 会展系统 应用于知名旅行社的会议及展览服务系统，具有高度定制能力，可以根据不同类型会展快速建设服务网站。
  • 通信服务 集成即时通、电子邮件等互联网通讯服务，支持移动、桌面和客厅电视系统。采用云服务的通信服务平台，更方便用户的业务信息管理。
  • 真伪认证 创新的基于云服务和移动设备的真伪认证服务，为构建诚信商业社会作出独特贡献。
  • 知识管理 创新的基于协同工作和社交服务的企业知识管理系统，为企业构筑无形资产的知识仓库。
  • 能力认证 支持政府、学校、企业等专业/职业技能培训、答疑和在线考核认证。
  • 电子商务 支持企业在线提供服务、设备租赁、产品销售以及分层直销。支持平台化加盟运作。
统一资源平台 构筑知行合一的系统应用

         统一资源平台 不仅可以帮助政府、企业等机构，快速建立内部的协同应用，形成协同办公、社交生活等，同时是开放的，可以自生长的平台，可以很好地方便第三方的二次扩展和开发。
